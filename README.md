# Install

+ It would only be necessary to install the dependencies. Finally you have to go to the /calculator-hooks folder and make the `npm install`.

```
/calculator-hooks (main)
npm i

```

+ Finally, run the process with `npm start` 

```
/calculator-hooks (main)
npm start

```
